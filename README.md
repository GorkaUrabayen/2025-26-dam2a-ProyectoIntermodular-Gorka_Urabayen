
# TINY-DEFENSE - Tower Defense Procedural
Este repositorio contiene el código fuente y los activos de un videojuego de género **Tower Defense** desarrollado en el motor Unity. El proyecto implementa un sistema de generación procedimental de niveles y una lógica de gestión de unidades defensivas.

## 📥 Instalación y Descarga

### Requisitos previos
* **Unity Hub** instalado.
* **Unity 2022.3.62f1 LTS** (o versión superior compatible).

### Pasos para descargar
1.  En la parte superior de este repositorio, haga clic en el botón verde **"Code"**.
2.  Seleccione la opción **"Download ZIP"** y extraiga el archivo en una carpeta de su elección.
3.  *(Alternativo)* Si dispone de Git instalado, puede clonar el repositorio mediante el siguiente comando en su terminal:
    ```bash
    git clone https://github.com/GorkaUrabayen/2025-26-dam2a-ProyectoIntermodular-Gorka_Urabayen.git
    ```
### Pasos para importar el proyecto
1. Clone el repositorio o descargue el código fuente:
    ```bash
    git clone [https://github.com/GorkaUrabayen/2025-26-dam2a-ProyectoIntermodular-Gorka_Urabayen.git](https://github.com/GorkaUrabayen/2025-26-dam2a-ProyectoIntermodular-Gorka_Urabayen.git)
    ```
2. Inicie **Unity Hub** e importe el proyecto mediante el botón "Add".
3. Abra el proyecto y localice la escena principal en la carpeta `Assets/Scenes` **EscenaInicio**.

### Nota sobre el rendimiento
Se recomienda encarecidamente utilizar la versión ejecutable (ZIP) para jugar, ya que garantiza que las velocidades de movimiento y las mecánicas de los enemigos funcionen de forma óptima, evitando las variaciones de rendimiento propias del modo de previsualización del editor de Unity.
